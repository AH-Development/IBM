Jeux de données et sources ouvertes
Dans ce monde axé sur les données, certains jeux de données sont librement accessibles à tous pour être consultés, utilisés, modifiés et partagés. Ces jeux de données sont appelés jeux de données ouverts.
Les jeux de données ouverts incluent une licence publique et sont très utiles pour votre parcours en tant que Data Scientist. Voici quelques-unes des sources de jeux de données ouverts les plus informatives.

https://www.data.gov/
https://www.census.gov/data.html
https://data.gov.uk/
https://www.opendatanetwork.com/
https://data.un.org/
